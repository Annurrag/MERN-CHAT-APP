**CHAT APP**

This is a full-stack real-time chat application built using the MERN stack with Socket.IO for instant messaging. The app supports one-to-one chats, group chats, typing indicators, message notifications, and live updates without page refresh. The UI is fully responsive, optimized for both mobile and desktop, and built using React + Tailwind CSS.

Users can search for other users, start conversations, create group chats, receive real-time notifications, and enjoy a smooth chat experience with automatically scrolling message windows and clean chat bubble alignment.

The backend is built with Node.js + Express, using JWT authentication, and MongoDB to store users, chats, and messages. Socket.IO enables real-time events such as sending messages, receiving messages, joining chat rooms, and live typing indicators.

Overall, the app delivers a seamless modern messaging experience similar to WhatsApp Web or Messenger, built end-to-end from scratch.
